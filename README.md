# Desert Extreme 🏜️

פרויקט מודרני לאתר טיולי מדבר, בנוי באמצעות **Astro**.

## 📁 מבנה הפרויקט המאורגן

- **`src/pages/`**: כל דפי האתר (הנתיבים). כולל תמיכה ברב-לשוניות.
- **`src/components/`**: רכיבי UI לשימוש חוזר (כמו Header ו-Footer).
- **`src/layouts/`**: תבניות דפים מרכזיות.
- **`src/i18n/`**: קבצי תרגום בפורמט JSON לניהול קל של טקסטים.
- **`src/scripts/`**: לוגיקה של JavaScript (Stripe, נגישות וכו').
- **`src/styles/`**: קבצי עיצוב CSS.
- **`public/`**: נכסים סטטיים (תמונות ווידאו) שמועברים ישירות לאתר.

## 🚀 פיתוח

1. התקנת dependencies: `npm install`
2. הרצת שרת פיתוח: `npm run dev`
3. בנייה לפרודקשן: `npm run build`

## 💳 מערכת תשלומים
האתר משלב את מערכת Stripe בצורה מאובטחת דרך `src/scripts/stripe-config.js`.
